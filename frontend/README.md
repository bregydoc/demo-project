# Frontend - Aesthetic Notes

Next.js 14 frontend for the Aesthetic Notes app.

## Setup

```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env to point to your backend API

# Run development server
npm run dev
```

## Building

```bash
npm run build
npm start
```

## Linting

```bash
npm run lint
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript (strict)
- Tailwind CSS
- TanStack Query
- Axios
- React Hook Form
